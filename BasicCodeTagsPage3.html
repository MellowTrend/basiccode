Let's get moving! How about some hyperlinks? With the "a" tag, you can create your own portal to go anywhere you wish! Notice the web address is placed in between quotation marks. The text in between the openning and closing tags describes the link.

<a href="MyWebPage.html">Go to "MyWebPage"</a>


But sometimes a picture is worth a thousand likes, so you can set an image as the link itself. Notice how the <img> is sandwitched in between the anchor tags.

<a href="BasicCodeTagsPage4.html" <img src=images/httpimage.jpg> </a>
