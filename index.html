Hey! Ever wonder how all that info on your tablet got there? It's all web-talk! The internet communicates with it's own language called HTML. Just like you have a language to express yourself, we use HTML code to tell the internet how we want words, images and links to appear on a web-page. How about you give it a try?

Ready...set...CODE!

Let's start simple. We use the <p> and </p>, known as the openning and closing "p" tags, to display letters and characters (think of p for paragraph). Check out the demo below:

<p>I am text squished between the openning and closing "p" tags!</p>






OMG! Here's an IMG tag, also known as the image tag. Notice how the source (src) points to a folder called "Images" in which the picture is located. : )

<img src="Images/Happyface.jpg />



The initial step to setting up your coding environment is to create and save your .html (that's, dot html) document. The document template should include the following tags, as well as the basic tags demonstrated above.

<!DOCTYPE!>
<html>
<head> </head>

<body>

</body>
</html>
